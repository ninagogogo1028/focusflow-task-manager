# FocusFlow Task Manager

FocusFlow 是一个智能化的个人任务管理工具，结合了看板、日历和 AI 辅助功能，旨在帮助用户高效管理工作与生活任务。

## 🌟 功能说明 (Features)

### 1. 多视图任务管理
- **Dashboard (仪表盘)**: 概览今日待办、Focus Score 状态、快速添加任务。
- **Kanban Board (看板)**: 经典的 To Do / In Progress / Done 工作流管理，支持拖拽任务流转。
- **Calendar (日历)**: 月视图查看任务分布，支持日期切换。
- **Archive (归档)**: 查看已完成任务，支持 "Keep Forever"（永久保存）或 7 天自动清理机制。

### 2. 🤖 AI 智能辅助
- **智能任务生成**: 拖拽文件到 Dashboard 即可通过 Gemini AI 自动分析并生成相关任务。
- **每日回顾 (Daily Recap)**: 智能生成每日工作总结和待办提醒。

### 3. 📱 移动端适配
- **响应式设计**: 完美适配手机端操作。
- **移动端专属交互**: 底部导航栏、优化的触摸输入体验。

### 4. 任务细节管理
- **任务分类**: 支持 Work / Personal 分类及颜色标识。
- **编辑与删除**: 支持任务名称的实时编辑和删除。
- **提醒功能**: 支持设置具体提醒时间。

---

## 💡 使用建议 (Usage Suggestions)

1. **日常工作流**:
   - 早上打开 **Dashboard** 查看 "Today's Priorities"。
   - 使用 **Kanban** 视图在工作进行中拖拽任务状态。
   - 完成任务后点击打钩，任务会自动进入 Archive。

2. **AI 自动捕获**:
   - 当你有通过文件（如文档、图片）产生的任务时，直接将其拖入 Dashboard 的 "Drop zone"，AI 会自动帮你起草任务标题和描述。

3. **归档管理**:
   - 系统默认会清理 7 天前的已完成任务。
   - 对于重要的里程碑任务，建议在 Archive 中开启 **"Keep Forever"**。

4. **手机端使用**:
   - 在局域网内，通过手机浏览器访问电脑 IP (如 `http://192.168.x.x:3000`) 即可随时随地管理任务。

## 🛠️ 本地运行 (Run Locally)

1. 安装依赖:
   ```bash
   npm install
   ```

2. 配置环境变量:
   在 `.env.local` 文件中填入你的 `GEMINI_API_KEY`。

3. 启动开发服务器:
   ```bash
   npm run dev
   ```

4. 构建生产版本:
   ```bash
   npm run build
   ```
